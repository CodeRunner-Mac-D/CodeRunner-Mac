## Overview

CodeRunner Professional Suite is a complete lightweight development environment that serves developers with the language breadth, debugging capability, and customization depth needed for fast, focused coding work without the overhead of a full IDE. It recognizes that not every coding task needs a project structure, dependency manager, and build system — sometimes you just need to write code and run it. The application's minimal resource footprint means it launches instantly and runs alongside heavier tools without competing for system resources.

Integrated debugging with breakpoints, variable inspection, and step-through execution provides real debugging capability for languages that support it, enabling investigation of logic errors without switching to a separate debugging tool or adding print statements. The debugger integrates naturally with the run workflow — set breakpoints, run, and inspect. Variable watches and call stack navigation provide the essential debugging views without the overwhelming interface of a full IDE debugger.

Custom run configurations define exactly how each file type is compiled and executed, enabling support for additional languages, custom toolchains, and specialized execution environments beyond the built-in language support. Configuration files specify the command, arguments, working directory, and environment variables for each language. Shared configurations can be exported for team consistency across developer workstations.

Snippet library stores reusable code fragments with tab-trigger expansion, making frequently needed patterns available with a few keystrokes without leaving the current file. Snippets support placeholder fields that tab between insertion points, and the library is organized by language. Custom snippets are created from selected code with a keyboard shortcut, building the library naturally from your own coding patterns.

CodeRunner Professional Suite is the development tool for programmers who value the ability to write and execute code immediately without ceremony, in a tool that respects the focused, iterative nature of scripting, prototyping, and exploratory coding. Its combination of instant execution, multi-language support, and genuine debugging capability makes it the lightweight editor that developers keep permanently in their dock.

## Performance & Integration

CodeRunner is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while CodeRunner processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
